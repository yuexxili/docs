## Display Types

| Model | Display |
|-------|---------|
| NQi (2016–2017) | Standard display |
| NQi Sharing (dual battery) | Black and white display |
| Most models | Model-specific LCD/LED display |

## Dashboard Removal

<Steps>
  <Step title="Access dashboard">
    On most models: remove the dashboard cover block (Section 3 in panel removal sequence).
  </Step>
  <Step title="Disconnect connectors">
    Note the position of all connectors before disconnecting. Photograph if needed.
  </Step>
  <Step title="Remove display unit">
    Unscrew mounting screws and carefully remove display. Avoid pulling on wiring.
  </Step>
</Steps>

## Failure Diagnosis

<AccordionGroup>
  <Accordion title="Display blank / not turning on">
    1. Check battery charge level — charge if flat
    2. Check fuse and main power connections
    3. Check display power connector — reseat
    4. If display remains blank with confirmed power supply, display unit may be faulty — replace
  </Accordion>
  <Accordion title="Display shows error code">
    Refer to [Diagnostic Codes](/repair/diagnostic-codes) for the full error code list and resolution steps.
  </Accordion>
  <Accordion title="Display flickering or intermittent">
    1. Inspect display connector — clean contacts and reseat
    2. Check wiring harness for damage or loose routing
    3. If flickering persists, replace display unit
  </Accordion>
  <Accordion title="Speedometer inaccurate">
    1. Check wheel speed sensor connector
    2. Inspect sensor mounting — ensure correct gap to magnet ring
    3. Replace sensor if faulty
  </Accordion>
  <Accordion title="Buttons not responding">
    1. Check switch connector and wiring
    2. Clean switch contacts if corroded
    3. Replace combination switch assembly if buttons are physically damaged
  </Accordion>
</AccordionGroup>

<Note>
  If display replacement is needed, [escalate to after-sales](/escalation) to confirm part compatibility for the specific model and year.
</Note>
