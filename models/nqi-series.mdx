## Known Failure Points

- BMS faults (OFFLINE = BMS Failure, MOS Failure) — see [Battery & BMS](/repair/battery-bms)
- HALL failure — see [Motor & Drivetrain](/repair/motor-drivetrain)
- MOSFET failure — see [Electrical & Controller](/repair/electrical-controller)
- Phase line short / default — see [Electrical & Controller](/repair/electrical-controller)
- Hydraulic disc brake failure — see [Brakes](/repair/brakes)
- Rear shock absorber failure — see [Suspension](/repair/suspension)

<Warning>
  If the issue is not resolved after following repair procedures, [escalate to after-sales](/escalation).
</Warning>
