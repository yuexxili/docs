## Known Failure Points

Refer to the **Trouble Shooting and Code List** section in the XQi3 manual for a full diagnostic code reference.

- Hydraulic disc brake failure — see [Brakes](/repair/brakes)
- Motor faults — see [Motor & Drivetrain](/repair/motor-drivetrain)
- Battery / BMS faults — see [Battery & BMS](/repair/battery-bms)
- Controller faults — see [Electrical & Controller](/repair/electrical-controller)
