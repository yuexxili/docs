## Controller Protection Modes

The NQi X FOC controller includes the following built-in protections:

- Current limit protection
- Rotation failure (overload) protection
- Under-voltage / over-voltage protection
- Power cut-off protection during charging or braking
- Control loss protection

## Known Failure Points

- ECU firmware — try updating ECU to latest version first
- Electric faucet lock failure — replace lock
- BCS failure — replace BCS
- HALL failure — see [Motor & Drivetrain](/repair/motor-drivetrain)
- MOSFET failure — see [Electrical & Controller](/repair/electrical-controller)
- Phase line short — see [Electrical & Controller](/repair/electrical-controller)
- Controller overtemperature — stop riding, allow to cool, then diagnose FOC

<Warning>
  If the issue is not resolved after following repair procedures, [escalate to after-sales](/escalation).
</Warning>
